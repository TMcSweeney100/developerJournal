# DevLog

DevLog is a full-stack software engineer diary app built with Next.js, Spring Boot, and PostgreSQL.

The purpose of DevLog is to help software engineers record what they worked on, what they learned, what blocked them, and what progress they made over time.

## Tech stack

Frontend:

- Next.js
- TypeScript
- shadcn/ui
- Tailwind CSS

Backend:

- Spring Boot
- Java
- Spring Data JPA
- PostgreSQL

## Project structure

```text
devlog/
  frontend/
  backend/
  AGENTS.md
  README.md
  TASKS.md
```

## Current goal

Get the frontend, backend, and database running.

## Local development database

For now, PostgreSQL will be installed and run locally.

Docker or a hosted PostgreSQL database can be considered later, but the first version will focus on learning how Spring Boot connects to a local PostgreSQL database.
